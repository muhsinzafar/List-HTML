03-Defination List

Defination: Definition list 'dl' is used for pairs of terms and their corresponding definitions.

Task: Provides Explanations of HTML, CSS & JavaScript.

HTML:
HTML(Hypertext Markup Language) is described as the foundational language used to create the structure and content of web pages. HTML consists of a series of elements that are used to enclose or wrap different parts of the content to make it appear or behave in a certain way. Each HTML element is represented by a tag, and these elements can contain text, images, links, and other content.
CSS:
CSS(Cascading Style Sheets) is described as the language used to control the presentation and layout of web pages. While HTML provides the structure and content, CSS is responsible for how the content is displayed, including aspects like colors, fonts, spacing, and the overall layout of the page.
JavaScript:
JavaScript(JS) is introduced as the programming language that adds interactivity and dynamic behavior to web pages. While HTML is used for structure and content, and CSS for presentation and layout, JavaScript enables the creation of features that respond to user actions, such as clicking buttons, filling out forms, or dynamically updating content without requiring a page reload.
